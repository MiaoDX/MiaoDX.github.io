🎉 彩蛋公告 ​

从 2026-03-11 起，本仓库的所有更新将使用 jj (Jujutsu) 进行版本控制，而非 raw git。

这是 LIP 的第一个彩蛋——不只是 Learn In Public，还要 Learn In Style。

为什么用 jj？ ​

  • 更简洁的 rebase 和 merge
  • 没有 index/staging 概念，commit 直接
  • 自动追踪操作历史，撤销更简单
  • 适合多 Agent 协作场景
